# Expense Tracker (MERN)

 This is a Full stack expense tracker made using [mongo db,express js, react js,node js]

## Usage

```
change config.env file in config folder
```

```
 npm install
 cd client npm install
 cd ..
 
 # Run front and backend
 npm run dev
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 
 # Build client
 cd client
 npm run build
 
 # Prod
 npm start
```
